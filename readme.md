# Logistics Control Tower Simulator (LCTS)

Simulador de uma Torre de Controle Logística desenvolvido em Python com FastAPI.

## Objetivo

Simular uma operação logística em tempo quase real, permitindo a construção de dashboards, indicadores operacionais e automações.

## Tecnologias

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Power BI

## Funcionalidades previstas

- Simulação de frota
- Simulação de entregas
- Eventos logísticos
- API REST
- Dashboard Power BI
- KPIs em tempo real
